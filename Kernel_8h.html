Detailed Description

File containing the definitions of Kernel and KernelProvider.

File containing the ReaDDy top level API.

This file contains the class definitions for Kernel and KernelProvider. A Kernel is used to execute Programs, i.e., instances of readdy::plugin::Program. The kernels can be built in or provided by shared libs in directories, which are loaded by the KernelProvider. Each Kernel has a readdy::plugin::Kernel::name by which it can be accessed in the KernelProvider.
