Detailed Description

Base class of kernels. A Kernel is used to execute Programs, i.e., instances of readdy::plugin::Program. The kernels can be built in or provided by shared libs in directories, which are loaded by the KernelProvider. Each Kernel has a #name by which it can be accessed in the KernelProvider.
