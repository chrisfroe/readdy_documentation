Detailed Description

Declaration of the base class of all order 1 potentials.

Declaration of the base class of all order 1 potentials. Subclasses have to implement calculateEnergy, calculateForce calculateForceAndEnergy, and replicate. The first three methods take a modifiable reference and a particle's position. The last method is for replication of the potential, so that it can be assigned to multiple particle types.
